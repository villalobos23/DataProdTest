Purpose of the tool

The purpose of this tool is to show how can shiny be used to handle interactive web based applications for statistical purposes, mainly the effect of variable selection in linear models and regression.

On the left side panel you have 4 options

  • Number of Cars:The amount of cars to be used to a total of 32
  • Characteristic to use (x) The variable to use as an independent variable
  • Characteristic to use (y) The variable to use as an dependent variable
  • Use Intercept? Whether to use an intercept or not for the analysis

On the right hand we have several tabs

  • Observations - Detailed info about the observations
  • Plot - plot of the fitted model
  • Residuals - Residual plots of the fitted model
  • Model Summary - Summary of the different values of the model
  • Data Summary - Summary of the Variables selected

Regarding plots and summaries, they may change according to the type of the variables selected, if it is a category variable, like Transmission(am) or Cylinders(cyl) the plot created is a boxplot instead of the expected line plot.Another thing to note is that the variables usted to be on the y-axis should be numeric variables to obtain understandable results

Dataset used

taken from the mtcars dataset R helpfile (?mtcars)

The data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973-74 models).

Variables

  1. mpg Miles/(US) gallon
  2. cyl Number of cylinders
  3. disp Displacement (cu.in.)
  4. hp Gross horsepower
  5. drat Rear axle ratio
  6. wt Weight (lb/1000)
  7. qsec 1/4 mile time in seconds
  8. vs V/S V engine or Straight engine
  9. am Transmission Automatic or Manual
  10. gear Number of forward gears
  11. carb Number of carburetors
